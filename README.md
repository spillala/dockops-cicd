# DockOps CI/CD

CI/CD pipeline templates for autonomous drone software teams.

Built for drone companies using PX4 / ArduPilot on Kubernetes.

## What's Inside
- GitHub Actions workflows for drone app CI/CD
- Firmware build and signing pipelines
- K8s deployment with automated rollback
- OTA update pipeline templates

## Stack
- Kubernetes (K3s)
- ArgoCD for GitOps
- GitHub Actions
- Claude API for AI-assisted verification
